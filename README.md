# Task Manager 🚀

This is a simple Task Manager project built using FastAPI and Streamlit.

## Features
- Add Task
- Delete Task
- Update Task
- View Tasks

## Tech Stack
- FastAPI
- Streamlit
- Python

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run FastAPI:
uvicorn app.main:app --reload

3. Run Streamlit:
streamlit run app/app.py
